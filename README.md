Flask web app to run as a dashboard on my pi zero 2 with a 15.6in low power display. Created by Claude.

Displays;
date, day, local and utc time
weather; current conditions, 12 hr outlook, and 7 day outlook
stocks; can add, edit, remove symbols in admin panel
photos; rotating photos, selects from a pictures folder on the pi
reminders; can add, edit, remove reminders in admin panel
spanish; rotates an array of 1000 spanish words, english definitions, example

admin panel; /admin does not need login or password. just for editing stocks and reminders
